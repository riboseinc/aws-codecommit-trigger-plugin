Name or URL of the AWS SQS queue to monitor. The plugin will trigger a new build when a matching branch update is posted to this queue.

Configure Jenkins to monitor an AWS SQS queue for CodeCommit updates. Jenkins will start a new build if a matching branch update is posted to the SQS queue.

The queue will only be monitored if you have configured at least one job to listen for this queue. See the build trigger configuration of individual jobs.

Further documentation available here: AWS Codecommit Trigger Plugin.
